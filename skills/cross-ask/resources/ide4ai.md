# Cross Ask — ide4ai 问卷模板

> 当其他项目工程师需要向 IDE MCP 工具工程师提问时，使用此模板。

## 项目职责

ide4ai 提供 AI IDE 工具集（Python），包括代码导航、编辑、LSP 集成和终端操作等 MCP 工具，供 AI Agent 在开发环境中执行代码任务。

## 问卷必填字段

### 1. 涉及的工具类别

| 类别 | 说明 |
|------|------|
| **代码导航** | 文件浏览、符号搜索、定义跳转 |
| **代码编辑** | 文件创建/修改/删除、批量操作 |
| **LSP 集成** | Language Server Protocol 功能（补全、诊断、重构） |
| **终端操作** | 命令执行、输出捕获 |

### 2. MCP 工具信息

如问题涉及具体 MCP 工具：
- 工具名称
- 参数 schema 和返回结构
- 工具的前置条件和副作用

### 3. 期望回答格式

| 问题类型 | 期望回答 |
|---------|---------|
| 工具定义 | 工具名 + 参数 + 返回结构 + 使用示例 |
| LSP 集成 | 支持的 LSP 特性 + 配置方式 |
| 错误处理 | 错误码 + 恢复策略 |

## 常见问询场景

| 发起方 | 典型问题 |
|--------|---------|
| client | IDE 工具的启动参数和运行时依赖？ |
| python-sdk | MCP 工具的 schema 和调用约定？ |
| protocol | IDE 工具作为 Computer 组件的注册方式？ |
| office4ai | 与 IDE 工具在同一 Computer 中共存时的工具命名冲突处理？ |
